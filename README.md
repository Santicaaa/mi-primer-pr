# Mi Primer PR

Un repositorio de práctica para aprender el flujo de trabajo con Git y Pull Requests.

## ¿Qué hay acá?

- `saludar.py`: un script mínimo que imprmie un saludo.

## Uso

```bash
python3 saludar.py
```

## Objetivo

Este proyecto existe para practicar el ciclo completo de una contribución:
crear una rama, hacer un cambio, subirlo y abrir un Pull Request.
